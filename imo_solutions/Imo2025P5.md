## Problem

Alice and Bazza are playing the *inekoalaty game*, a two-player game whose rules depend on a positive real number $\lambda$. On the $n$th turn (starting with $n=1$):
- If $n$ is odd, Alice chooses $x_n \geq 0$ such that $x_1 + \cdots + x_n \leq \lambda n$.
- If $n$ is even, Bazza chooses $x_n \geq 0$ such that $x_1^2 + \cdots + x_n^2 \leq n$.

If a player cannot choose, the game ends and the other player wins. If the game goes on forever, neither wins.

Determine all values of $\lambda$ for which Alice has a winning strategy and all those for which Bazza has a winning strategy.

---

## Solution

**Answer:** Alice wins for $\lambda > 1/\sqrt{2}$; Bazza wins for $\lambda < 1/\sqrt{2}$; neither wins for $\lambda = 1/\sqrt{2}$.

**Alice's strategy for $\lambda \geq 1/\sqrt{2}$:** Alice always plays $x_{2i+1} = 0$.

With this strategy, on turn $n = 2k+1$:
$$\sum_{j=1}^{2k} x_j = x_2 + x_4 + \cdots + x_{2k} \leq k \cdot \sqrt{\frac{x_2^2+\cdots+x_{2k}^2}{k}} \leq k \cdot \sqrt{\frac{2k}{k}} = \sqrt{2} \cdot k < \lambda(2k+1),$$
so Alice can always play $x_{2k+1} = 0$. She never loses.

If $\lambda > 1/\sqrt{2}$: for large $k$, $\lambda(2k+1) - \sqrt{2}k \to \infty$, so Alice can play a large $x_{2k+1}$, forcing $\sum x_j^2 > 2(k+1)$ on Bazza's next turn. Alice wins.

**Bazza's strategy for $\lambda \leq 1/\sqrt{2}$:** Bazza always plays $x_{2i+2} = \sqrt{2 - x_{2i+1}^2}$ (the maximum allowed).

With this strategy, on Alice's turn $2k+1$:
$$\lambda(2k+1) \geq x_1 + x_2 + \cdots + x_{2k+1} = \sum_{i=0}^{k-1}(x_{2i+1} + \sqrt{2-x_{2i+1}^2}) + x_{2k+1} \geq \sqrt{2} \cdot k + x_{2k+1},$$
using $t + \sqrt{2-t^2} \geq \sqrt{2}$ for $0 \leq t \leq \sqrt{2}$. So $x_{2k+1} \leq \lambda(2k+1) - \sqrt{2}k$.

Bazza can always play since $x_1^2+\cdots+x_{2k}^2 + x_{2k+1}^2 \leq 2k + x_{2k+1}^2 < 2(k+1)$ iff $x_{2k+1} < \sqrt{2}$, which holds since $x_{2k+1} \leq \lambda(2k+1) - \sqrt{2}k < \sqrt{2}$ for $\lambda \leq 1/\sqrt{2}$.

If $\lambda < 1/\sqrt{2}$: for large $k$, $\lambda(2k+1) - \sqrt{2}k < 0$, so Alice cannot play. Bazza wins.

**$\lambda = 1/\sqrt{2}$:** Alice can always play $x_{2k+1} = 0$ (maintaining the sum constraint), and Bazza can always respond. Neither player wins. $\blacksquare$
