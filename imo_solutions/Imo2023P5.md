## Problem

Let $n$ be a positive integer. A *Japanese triangle* consists of $1+2+\cdots+n$ circles arranged in an equilateral triangular shape such that for each $1 \leq i \leq n$, the $i$th row contains exactly $i$ circles, exactly one of which is colored red. A *ninja path* in a Japanese triangle is a sequence of $n$ circles obtained by starting in the top row, then repeatedly going from a circle to one of the two circles immediately below it and finishing in the bottom row.

In terms of $n$, find the greatest $k$ such that in each Japanese triangle there is a ninja path containing at least $k$ red circles.

---

## Solution

**Answer:** $k = \lfloor \log_2 n \rfloor + 1$.

**Construction (upper bound):** It suffices to find a Japanese triangle for $n = 2^e - 1$ such that at most $e$ red circles appear in any ninja path.

Partition the rows into groups $\{1\}$, $\{2,3\}$, $\{4,5,6,7\}$, $\ldots$, $\{2^{e-1}, \ldots, 2^e - 1\}$. Place red circles so that within each group, the red circle in each row is chosen to "block" — specifically, any ninja path can pick at most one red circle from each group. (Place the red circle at the rightmost position in each row, so paths going left miss it and paths going right are forced to eventually go left.) This gives $e = \lfloor \log_2 n \rfloor + 1$ as the upper bound.

**Lower bound:** We show every Japanese triangle has a ninja path with at least $\lfloor \log_2 n \rfloor + 1$ red circles.

**Binary tree approach:** Construct a rooted binary tree $T_1$ on all circles. For each row (other than the bottom):
- Connect the red circle to both circles below it.
- White circles to the left of the red circle connect left.
- White circles to the right connect right.

Every leaf in $T_1$ is in the bottom row. Now build $T_2$ on just the red circles: each red circle is connected to the red circle directly below it in $T_1$ (if reachable through whites). Each red circle has at most 2 children in $T_2$.

Since $T_2$ is a rooted tree on $n$ nodes where each node has at most 2 children, its depth is at least $\lceil \log_2 n \rceil \geq \lfloor \log_2 n \rfloor$. A path from root to deepest leaf in $T_2$ corresponds to a ninja path hitting $\geq \lfloor \log_2 n \rfloor + 1$ red circles.

**Proof via sum $S_j$:** Let $f(i,j)$ = maximum red circles on a ninja path from row 1 to circle $(i,j)$. Define $S_j = \sum_{i=0}^j f(i,j)$.

**Claim:** $S_{j+1} \geq S_j + \lfloor S_j/j \rfloor + 1$.

*Proof.* Write $n = 2^c + r$ with $0 \leq r < 2^c$. By induction, $S_n \geq cn + 2r + 1 \geq (c+1)n \geq n(\lfloor \log_2 n \rfloor + 1)$. By pigeonhole on the $n$ circles in row $n$, some $f(i,n) \geq \lfloor \log_2 n \rfloor + 1$. $\blacksquare$
