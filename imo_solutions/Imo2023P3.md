## Problem

For each integer $k \geq 2$, determine all infinite sequences of positive integers $a_1, a_2, \ldots$ for which there exists a polynomial $P$ of the form
$$P(x) = x^k + c_{k-1}x^{k-1} + \cdots + c_1 x + c_0,$$
where $c_0, c_1, \ldots, c_{k-1}$ are non-negative integers, such that
$$P(a_n) = a_{n+1}a_{n+2}\cdots a_{n+k}$$
for every integer $n \geq 1$.

---

## Solution

**Answer:** All arithmetic progressions $a_n = d(n-1) + a_1$ for $d \geq 0$.

**Verification:** If $a_n = a_1 + d(n-1)$, then $a_{n+1} \cdots a_{n+k} = (a_n + d)(a_n + 2d)\cdots(a_n + kd)$, so take $P(x) = (x+d)(x+2d)\cdots(x+kd)$, which has non-negative coefficients.

**Converse:**

**Step 1: The sequence is non-decreasing or constant.**

From $P(a_{n-1}) = a_n a_{n+1} \cdots a_{n+k-1}$ and $P(a_n) = a_{n+1} \cdots a_{n+k}$, we get $a_{n+k} = \frac{P(a_n)}{P(a_{n-1})} \cdot a_n$.

Since $P$ is strictly increasing on $\mathbb{N}$, if $a_n < a_{n-1}$ then $a_{n+k} < a_n < a_{n-1}$, and by induction we get an infinite strictly decreasing sequence of positive integers — impossible. So the sequence is non-decreasing. If $a_n = a_{n-1}$ for any $n$, then $a_{n+k} = a_n$, and by downward induction the sequence is constant.

**Step 2: The differences are bounded.**

Let $C$ be a constant such that $P(x) < x^k + Cx^{k-1}$ for all $x \in \mathbb{N}$ (take $C = c_0 + c_1 + \cdots + c_{k-1} + 1$). Then:
$$a_{n+k} = \frac{P(a_n)}{a_{n+1}\cdots a_{n+k-1}} < \frac{a_n^k + Ca_n^{k-1}}{(a_n+1)(a_n+2)\cdots(a_n+k-1)} < a_n + C + 1.$$

So consecutive terms differ by at most $C+1$.

**Step 3: The differences eventually stabilize.**

Define $\Delta(n) = (a_{n+1}-a_n, a_{n+2}-a_{n+1}, \ldots, a_{n+k}-a_{n+k-1})$. This tuple takes values in $\{0, 1, \ldots, C+1\}^k$, so finitely many values. Some tuple $\mathbf{d} = (d_1, \ldots, d_k)$ appears infinitely often.

For any $N$ where $\Delta(N) = \mathbf{d}$:
$$P(a_N) = (a_N + d_1)(a_N + d_1+d_2)\cdots(a_N + d_1+\cdots+d_k).$$

This polynomial identity holds for infinitely many $a_N$ (since the sequence is unbounded if non-constant), so:
$$P(X) = (X + d_1)(X + d_1+d_2)\cdots(X+d_1+\cdots+d_k) \text{ as polynomials.}$$

This uniquely determines $P$, hence uniquely determines the tuple $\mathbf{d}$ (exactly one tuple can appear infinitely often). Thus $\Delta(n)$ is eventually constant: $a_{n+1} - a_n = d$ for all large $n$ and some fixed $d \geq 0$.

**Step 4: The sequence is arithmetic from the start.**

Once $\Delta$ is eventually constant with common difference $d$, and $P(X) = (X+d)(X+2d)\cdots(X+kd)$, one can verify by downward induction that $a_{n+1} - a_n = d$ for all $n$. $\blacksquare$
