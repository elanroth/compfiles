## Problem

There are $4n$ pebbles of weights $1, 2, 3, \ldots, 4n$. Each pebble is coloured in one of $n$ colours, and there are four pebbles of each colour. Show that we can arrange the pebbles into two piles so that:
- the total weights of both piles are equal, and
- each pile contains two pebbles of each colour.

---

## Solution

**Key observation:** $1 + 4n = 2 + (4n-1) = 3 + (4n-2) = \cdots$

Pair up the pebbles: for each $k = 1, 2, \ldots, 2n$, tape a string between pebble $k$ and pebble $4n+1-k$. Each string connects two pebbles with the same sum $4n+1$.

Place all four pebbles of the same colour in a box (yielding $n$ boxes total). Each box contains exactly 4 pebbles, hence exactly 2 strings. A string between two pebbles in the **same** box counts as a self-loop (degree 2 contribution to that vertex).

**Reformulation.** The $2n$ strings form a multigraph on $n$ vertices (the boxes), where each vertex has degree exactly 4 (since each box has 4 pebbles, hence 4 string-endpoints). We need to 2-color the strings (blue/green) so that each vertex has exactly 2 blue and 2 green strings. (A string in pile A gets color blue; pile B gets green. The color condition ensures each pile has exactly 2 pebbles of each color, and the paired-sum construction ensures equal total weights.)

**Solution of reformulation.** We need to 2-color the edges of a 4-regular multigraph so each vertex has exactly 2 edges of each color.

Decompose each connected component into an Eulerian circuit (possible since 4 is even and the graph is connected within each component). An Eulerian circuit in a component with $k$ vertices has $2k$ edges. Color the edges alternating blue-green along the circuit — since $2k$ is even, the coloring closes up consistently, giving each vertex exactly 2 blue and 2 green edges.

This coloring assigns each string a color, i.e., assigns each pair of pebbles $\{k, 4n+1-k\}$ to either pile A (blue) or pile B (green). By construction:
- Each pile gets exactly one pebble from each string, so both piles have $2n$ pebbles total.
- Total weight of each pile: $\sum_k k = n(2n+1) \cdot 2 / 2 = n(2n+1)$ — both piles have equal weight $n(2n+1)$.
- Each colour class has 4 pebbles split as 2+2 between piles, since each box has 2 blue and 2 green edges. $\blacksquare$
