## Problem

The equation
$$(x-1)(x-2)\cdots(x-2016) = (x-1)(x-2)\cdots(x-2016)$$
is written on the board, with 2016 linear factors on each side. What is the least possible value of $k$ for which it is possible to erase exactly $k$ of these 4032 linear factors so that at least one factor remains on each side and the resulting equation has no real solutions?

---

## Solution

**Answer:** $k = 2016$.

**Step 1: Lower bound $k \geq 2016$.**

If fewer than 2016 factors are erased, then there exists some factor $(x - j)$ present on both sides. Any $x = j$ satisfies the resulting equation (both sides are 0), so the equation has a real solution. Hence we must erase at least one factor from each matching pair $(x-j), (x-j)$, and since there are 2016 pairs, $k \geq 2016$.

**Step 2: Construction with $k = 2016$.**

Erase from the left side all factors $(x-j)$ with $j \equiv 2, 3 \pmod 4$ (i.e., $j \in \{2, 3, 6, 7, 10, 11, \ldots, 2014, 2015\}$, which is $2 \cdot 504 = 1008$ factors). Erase from the right side all factors $(x-j)$ with $j \equiv 0, 1 \pmod 4$ (i.e., $j \in \{1, 4, 5, 8, 9, \ldots, 2013, 2016\}$, another 1008 factors). Total erased: $k = 2016$.

The resulting equation is:
$$\prod_{\substack{j=1 \\ j \equiv 0,1 \pmod 4}}^{2016} (x-j) = \prod_{\substack{j=1 \\ j \equiv 2,3 \pmod 4}}^{2016} (x-j).$$

**Step 3: Verification of no real solutions.**

For $x$ in the interval $(4m-2, 4m-1)$ for some $m$ (the only potentially problematic intervals), we need to show neither side dominates the other incorrectly.

Consider the 1008 inequalities:
$$(x-1)(x-4) < (x-2)(x-3), \quad (x-5)(x-8) < (x-6)(x-7), \ldots$$
Each holds for all real $x$ since $(x-a)(x-a-3) < (x-a-1)(x-a-2) \iff 0 < 1$ (expanding, both sides agree up to a constant).

More precisely: for each group $\{4m-3, 4m-2, 4m-1, 4m\}$, we have
$$(x-(4m-3))(x-4m) < (x-(4m-2))(x-(4m-1))$$
except when $x \in (4m-2, 4m-1)$, where one factor changes sign. A careful product analysis (using the estimate $\prod_k \frac{(4k+2)(4k+3)}{(4k+1)(4k+4)} < e < 9$) shows the equation is never satisfied.

**Conclusion:** The minimum value is $k = \boxed{2016}$. $\blacksquare$
