## Problem

Let $n \geq 3$ be an integer. We say that an arrangement of the numbers $1, 2, \ldots, n^2$ in an $n \times n$ table is *row-valid* if the numbers in each row can be permuted to form an arithmetic progression, and *column-valid* if the numbers in each column can be permuted to form an arithmetic progression.

For what values of $n$ is it possible to transform any row-valid arrangement into a column-valid arrangement by permuting the numbers in each row?

---

## Solution

**Answer:** $n$ prime only.

**Proof for $n = p$ prime:**

In any arithmetic progression of $p$ terms, either every term has a different residue modulo $p$ (if the common difference is not divisible by $p$), or all terms are congruent modulo $p$ (if the common difference is divisible by $p$).

Look at the multiples of $p$ in a row-valid $p \times p$ table. There are exactly $p$ multiples of $p$ in $\{1, \ldots, p^2\}$.

**Case 1:** All $p$ multiples of $p$ lie in the same row. Then each row's arithmetic progression has common difference divisible by $p$, so each row consists of the elements $\equiv k \pmod p$ for some $k$. We can permute each row so that the $j$th column contains exactly the elements $\equiv j \pmod p$, forming arithmetic progressions in each column.

**Case 2:** All $p$ multiples of $p$ lie in different rows. Then each row's AP has common difference not divisible by $p$, so each row contains every residue class mod $p$ exactly once. Permute each row so the $j$th column contains the $j \bmod p$ elements — these form arithmetic progressions.

**Counterexample for $n$ composite:**

Let $p$ be any prime divisor of $n$. Construct the following table:
- Row 1 contains $1$ through $n$.
- Rows 2 through $p+1$ contain $p+1$ to $np+n$, partitioned into $p$ APs with common difference $p$.
- Remaining rows contain the rest in reading order.

In this row-valid arrangement, for any column-valid permutation of rows, the column containing entry $2$ must also contain $n^2 - n + 2$ (since column entries span the full range and must form an AP). But $n+2$ and $2n+2$ lie in the same row (rows 2–$p+1$ are grouped), and any column-valid permutation would require them to be in different columns — contradiction.

Hence for $n$ composite, there exists a row-valid arrangement with no column-valid transformation. $\blacksquare$
