## Problem

Find all functions $g : \mathbb{Z}_{>0} \to \mathbb{Z}_{>0}$ such that
$$(g(m) + n)(g(n) + m)$$
is always a perfect square.

---

## Solution

**Answer:** $g(n) = n + c$ for some constant $c \geq 0$.

It is easy to verify these work (the product equals $(m+n+c)^2$). We prove these are the only solutions.

**Step 1: Key modular claim.**

*Claim:* $g(n) \equiv g(n')\ (\text{mod}\ p) \Rightarrow n \equiv n'\ (\text{mod}\ p)$ for any prime $p$.

*Proof:* Suppose $g(n) \equiv g(n') \pmod p$. Pick a large integer $M$ such that $\nu_p(M + g(n))$ and $\nu_p(M + g(n'))$ are both odd (possible since $\nu_p$ of $M + g(n)$ as $M$ varies eventually takes all values). Since $(M + g(n))(n + g(M))$ and $(M + g(n'))(n' + g(M))$ are both perfect squares, and the first factors have odd $p$-adic valuation, we need the second factors to also have odd $p$-adic valuation. Hence $\nu_p(n + g(M)) \equiv 1 \pmod 2$ and $\nu_p(n' + g(M)) \equiv 1 \pmod 2$, giving $p \mid n + g(M)$ and $p \mid n' + g(M)$, so $n \equiv n' \equiv -g(M) \pmod p$. $\square$

**Step 2: $g$ has slope $\pm 1$.**

Applying the claim with $p$ any prime: $g$ is injective (setting $n = n'$ gives nothing; but if $g(n) = g(n')$ then $n \equiv n' \pmod p$ for all primes $p$, so $n = n'$).

Moreover, $|g(n+1) - g(n)| = 1$ for all $n$ (the claim forces $g(n+1) \not\equiv g(n) \pmod p$ iff $(n+1) \not\equiv n \pmod p$, i.e., $p \nmid 1$, which is always true; hence consecutive values of $g$ are non-congruent mod every prime, forcing them to differ by exactly 1 in absolute value).

**Step 3: Concluding $g(n) = n + c$.**

Since $g$ maps positive integers to positive integers, is injective, and $|g(n+1) - g(n)| = 1$, $g$ must be linear with slope $+1$ (slope $-1$ would eventually give non-positive values). So $g(n) = n + c$ for some $c \geq 0$. $\blacksquare$
